# URL-Shortener
A modern and responsive URL Shortener web application that transforms long, complex URLs into short, easy-to-share links. The project features a clean and user-friendly interface, allowing users to quickly generate shortened URLs with just a few clicks. It is designed to improve link sharing, enhance readability..





✨ Features
🔗 Convert long URLs into short links instantly
📋 Copy shortened URLs with one click
⚡ Fast API integration
📱 Fully responsive design
❌ Input validation and error handling
🎨 Clean and modern user interface
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
TinyURL API (or your chosen URL Shortener API)

🚀 Getting Started
1. Clone the repository
git clone https://github.com/anand98prakash/URL-Shortener.git
2. Navigate to the project folder
cd URL-Shortener
3. Open the project

Simply open index.html in your browser, or use the VS Code Live Server extension for a better development experience.

📖 How to Use
Open the application.
Enter a valid long URL.
Click the Shorten URL button.
Copy and share the generated short link.
